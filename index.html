Daily Research System

Macro Signals Daily

A public-facing daily macro intelligence demo that monitors financial headlines, classifies narrative shifts using a transparent rules-based engine, and checks whether market proxies confirm the story. Built as a lightweight alternative to expensive always-on LLM API pipelines.

  • Reads public RSS feeds and filters for macro-relevant signals.
  • Scores themes like policy, inflation, growth, liquidity, and risk.
  • Cross-checks narrative against market proxies like SPY, TLT, UUP, USO, and HYG.
  • Publishes a client-friendly daily HTML brief automatically.

Browser Audio Analysis

Trump Reference Phrase Comparison

A static portfolio demo that compares a short browser recording against selected Donald Trump reference phrase clips. The app extracts acoustic features in the frontend and returns transparent similarity scores for pitch/prosody, rhythm, energy, timing, pauses, and phrase/content match.

  • Runs entirely in the browser with no backend, database, or ML service.
  • Uses the Web Audio API and MediaRecorder to process reference and user audio.
  • Breaks results into explainable score cards instead of a single black-box match.
  • Clearly documents that it is not transcription, biometrics, or identity verification.
